＊＊＊初めに＊＊＊

　ロックマンXDiVEの二次創作ゲームです。
　ただし、ストーリーはほぼありません。
　３時間くらいで終わると思います。


　＊＊＊操作方法＊＊＊

　　上下左右：移動
　　Z：決定、メニュー画面
　　X：キャンセル
　　Shift：ダンジョン探索中、マップ表示

　実況や動画配信などは禁止です。


　＊＊＊他＊＊＊

　・クエスト：常設は何度でも挑戦可。常設と書いていないものは一回だけです。
　　　　　　　クエスト覧の下から２つめは強め、一番下はそのダンジョンボス並の強さです。
　　　　　　　暗闇の迷宮は下３つは通常ダンジョンボスより強いです（ラスボスよりは弱め）。
　・アイテム：交換した装備前提の難易度になってます。なってしまいました。
　　　　　　　おすすめはアサシンダガーです。むしろこれなかったら中盤は縛りプレイになりえます。
　・バランス：序盤に地下迷宮で素材集めしないと割ときついと思います。
　　　　　　　テストプレイしてたらもうよくわかんなくなってきた。助けて
